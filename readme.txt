A starter theme called Gulpy

== Description ==

Make WP theme with Gulp to help development tasks.

== Installation ==

1. Download the github repository in your wp-content/themes :
`git clone https://github.com/valerieblanchard/gulpy.git`
2. In your admin panel, go to Appearance > Themes
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Gulpy includes support for Infinite Scroll in Jetpack.
